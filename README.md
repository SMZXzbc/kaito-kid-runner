## 🚀 快速开始

### 环境要求
- Python 3.10 或更高版本
- Pygame 2.x

### 安装依赖
```bash
pip install pygame

运行游戏
python dino_game.py
🎮 操作说明
表格
按键	功能	说明
↑ / 空格	跳跃	按两次可实现二段跳
↓	下蹲	躲避高处障碍
← / →	选择道具	在底部道具栏切换
Enter	使用道具	使用当前选中的道具
P	暂停/继续	游戏过程中随时暂停
R	重新开始	死亡后按 R 复活
ESC	退出游戏	直接关闭程序
进阶操作
二段跳后滑翔：完成二段跳后，按住 空格 或 ↑ 键可进入滑翔状态，缓慢下落并加速
重力区操作：进入紫色重力反转区域后，跳跃方向会反转，需要适应反向操作
✨ 游戏特色
🎯 六阶段难度系统
表格
阶段	名称	时长	障碍类型
1	教学阶段	15秒	无障碍，熟悉操作
2	尖刺阶段	40秒	地面尖刺（低/中/高）
3	子弹阶段	80秒	尖刺 + 横向子弹（3秒预警）
4	导弹阶段	140秒	尖刺 + 子弹 + 追踪导弹
5	重力阶段	220秒	全部障碍 + 重力反转区域
6	地狱模式	无限	所有障碍混合，速度持续提升
🛡️ 道具系统
表格
道具	图标	效果	持续时间
护盾	🔵 S	抵挡一次致命伤害	被动触发
时停	🔵 时钟	冻结所有障碍移动	2秒
缩小药水	🟣 瓶子	角色缩小50%，更容易躲避	5秒
火箭靴	🔴 火箭	速度翻倍，撞毁尖刺和子弹	3秒
扑克牌	🟡 箭头	向前发射扑克牌，摧毁子弹	即时
🌙 昼夜交替系统
每 30 秒切换一次白天/黑夜
夜晚模式下障碍和背景颜色会变化，增加视觉挑战
❤️ 新手保护机制
前两次死亡自动免费复活（金色特效提示）
后续死亡可通过拾取 心形钥匙 道具复活
复活后 2 秒无敌时间
